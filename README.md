Konsep OOP yang Digunakan :
- Class
Class yang dibuat adalah Transaksi.

- Object
$transaksi = new Transaksi($mysqli);

- Method
  - getOrders() → mengambil data transaksi
  - updateStatus() → mengubah status transaksi
  - getDetail() → mengambil detail transaksi

- Encapsulation
Variabel koneksi database disimpan dalam property:
private $db;
sehingga tidak bisa diakses langsung dari luar class.

Kesimpulan
Dengan menggunakan konsep Object Oriented Programming (OOP), kode program
menjadi lebih terstruktur, mudah dipelihara, dan dapat digunakan kembali. Pada
tugas ini fitur transaksi diubah menjadi class Transaksi yang memiliki beberapa
method seperti getOrders(), updateStatus(), dan getDetail().
